# 2026年06年28日

今日正式起草该项目主页`index.html`文件。大量工作都是豆包完成的！
`ui`大部分致敬（**模仿**）了`htmls.dev`，因为几、`js`个站看下来，就这个站很顺眼。
整体思路，制作一个`indx`首页做导航页，主要组成部分就是网页头部区域、搜索栏区域、标签栏按键区域、大卡片区域区域、网站底部区域。确定了之后就开始和豆包一起吧ui做出来，做出来大概就是现在的样子，于是开始完善各个区域的功能。
1、首先是网站头部，需要维护的地方非常少，可修改的数据很少，无非就是主页名称、描述都是一次性修改玩就完事了。右上角的导航网址按键也是一样，此部分不多说
2、下方的搜索栏逻辑也很简单，就是在点击搜索后，下面的标签分类后再筛选剩下的大卡片信息，主要是名字、描述、标签区域的索引，满足条件的显示，比如桌本站默认是全部分类的显示，再索引显示区域满足条件的标签，如果选择使用工具，那么会再此部分显示的大卡片进行判断再筛选显示。
3、分类标签按键，此部分的按键自动根据数据条目及属性进行显示。    `const toolList = [` ，判断数量，根据`catName` 数据增加对应按键，根据数量显示分类数量。这是以后主要维护的部分。
