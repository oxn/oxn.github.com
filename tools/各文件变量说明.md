#  特殊符号大全 `symbols-copy-and-paste.html`
```html
<!--
=============================================
特殊符号大全工具 - 数据驱动版
维护说明：
1. 页面所有分类、符号网格全部由 JS 中 const symData 对象自动生成
   HTML 内无任何手写符号方块，后续仅需维护 symData 即可，无需改动页面DOM
2. symData 数据结构规范：
   {
     分类标识: {
        title: "分类显示名称（页面顶部标签、区块标题）",
        list: [
            "一串连续符号字符串1",
            "一串连续符号字符串2"
        ]
     }
   }
3. 维护操作指南：
   ① 新增符号：找到对应分类 list 字符串，直接追加字符，刷新页面自动生成格子
   ② 新增分类：在 symData 内新增一组键值对，页面自动生成顶部标签+符号区块
   ③ 删除分类：直接删除对应分类整段键值对，页面自动移除标签与区块
   ④ 修改已有符号：编辑对应 list 内的字符，增删改后刷新生效
4. 内置功能：
   - 全局明暗主题，与cytools站点统一存储配置
   - 分类筛选、全部展示、关键词搜索匹配符号/分类
   - 点击符号自动复制剪贴板，拼接至下方文本框
   - 一键清空拼接内容、一键复制全部拼接文本
   - 移动端自适应布局，符号网格自动适配屏幕宽度
5. 页面结构固定顺序，无需调整：
   顶部导航栏 → 页面标题区域 → 搜索栏 → 拼接文本框 → 分类标签栏 → 自动渲染符号区块 → 底部三栏工具介绍卡片
6. 注意事项：
   - list 内仅填写 Unicode 符号，不要换行、空格干扰字符拆分
   - 字符顺序严格按字符串从左到右、从上到下自动排列网格
   - 搜索功能依赖 symData 完整字符，不可清空对应分类 list
=============================================
-->
```
