<!--
=============================================
特殊符号大全工具 - 数据驱动完整版
维护说明：
1. 页面所有分类标签、符号网格全部由 JS 中 const symData 对象自动生成
   HTML 内无任何手写符号方块、无手写分类标签，后续仅需维护 symData 即可，无需改动页面DOM
2. symData 数据结构规范：
   {
     分类标识: {
        title: "分类显示名称（页面顶部标签、区块标题）",
        singleItem: true/false 【可选参数】
        list: [
            "一串连续符号字符串1",
            "一串连续符号字符串2"
        ]
     }
   }
   singleItem = true：数组每一项完整一格，不拆分字符（颜文字专用）
   singleItem = false / 不写：字符串逐个字符拆分格子（普通数学/字母符号，原有逻辑）
3. 维护操作指南：
   ① 新增符号：找到对应分类 list 字符串，直接追加字符，刷新页面自动生成格子
   ② 新增分类：在 symData 内新增一组键值对，页面自动生成顶部标签+符号区块
   ③ 删除分类：直接删除对应分类整段键值对，页面自动移除标签与区块
   ④ 修改已有符号：编辑对应 list 内的字符，增删改后刷新生效
4. 内置功能：
   - 顶部渐变Toast复制成功弹窗，浅色/暗黑双主题适配，2秒自动淡出
   - 全局明暗主题，修复切换失效问题，与cytools站点统一存储配置
   - 「全部」固定首标签，点击显示所有分类；单标签筛选对应区块
   - 关键词搜索匹配符号/分类标题，模糊检索
   - 点击符号自动复制剪贴板，拼接至下方文本框
   - 一键清空拼接内容、一键复制全部拼接文本
   - 移动端自适应布局，符号网格自动适配屏幕宽度
5. 页面结构固定顺序，无需调整：
   顶部导航栏 → 页面标题区域 → 搜索栏 → 拼接文本框 → 自动渲染分类标签栏 → 自动渲染符号区块 → 底部三栏工具介绍卡片 → Toast弹窗容器
6. 注意事项：
   - 普通分类不填 singleItem，沿用原有逐字符拆分逻辑
   - 颜文字等需要整块展示的分类添加 singleItem: true
   - 搜索、弹窗、筛选功能全部依赖 symData 完整字符，不可清空对应分类 list
=============================================
-->
